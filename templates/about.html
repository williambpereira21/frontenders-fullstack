Sobre

Este é um projeto didático desenvolvido em Python com o microframework Flask e banco de dados SQLite.

O sistema permite que usuários autenticados com uma conta Google criem, editem e compartilhem pequenos blocos de notas chamados Pads. A proposta é apresentar, na prática, conceitos de autenticação, CRUD, templates, rotas e organização de aplicações Web.

O desenvolvimento faz parte das atividades da turma 2025.2 de Python do Senac Rio / Campo Grande, sendo continuamente aprimorado conforme avançamos nos conteúdos do curso.

Tecnologias usadas

  • Python 3 — linguagem base do projeto.
  • Flask — microframework Web.
  • SQLite — banco de dados leve e integrado.
  • Jinja2 — engine de templates do Flask.
  • Bootstrap 5 — layout e componentes responsivos.
  • Google OAuth 2.0 — autenticação de usuários.
  • HTML5, CSS3 e JavaScript — estrutura e interação da camada Web.
